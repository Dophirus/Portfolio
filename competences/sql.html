SQL

Compétence Technique

Vue d'ensemble

SQL (Structured Query Language) est un langage de programmation conçu pour la gestion et la manipulation de bases de données relationnelles. Il permet d'effectuer des opérations telles que la création, la lecture, la mise à jour et la suppression de données. La maîtrise de SQL est cruciale pour le développement d'applications qui nécessitent le stockage et la récupération efficace d'informations.

Développement personnel

J'ai développé mes compétences en SQL à travers divers projets, en apprenant à concevoir des schémas de base de données, à écrire des requêtes complexes et à optimiser les performances des requêtes. J'ai également exploré différentes fonctionnalités de SQL, telles que les transactions, les procédures stockées et les déclencheurs.

Application de la compétence

J'ai appliqué mes compétences en SQL dans les contextes suivants :

  • Conception de bases de données : j'ai créé des bases de données pour stocker les données des applications, en définissant les tables, les colonnes, les relations et les contraintes.
  • Manipulation de données : j'ai écrit des requêtes SQL pour insérer, lire, mettre à jour et supprimer les données des bases de données.
  • Requêtes complexes : j'ai élaboré des requêtes SQL avancées pour effectuer des opérations de filtrage, de tri, de regroupement et de jointure sur les données.
  • Optimisation des performances : j'ai optimisé les requêtes SQL pour améliorer la vitesse d'exécution et réduire la charge sur les bases de données.
  • Intégration avec les applications : j'ai intégré le code SQL dans les applications pour permettre l'accès et la manipulation des données.

Exemples concrets

Projet Baryton (application de quiz et blind-tests pour bars) :

  • J'ai utilisé SQL pour créer une base de données qui stocke les informations relatives aux quiz (questions, réponses, catégories) et aux blind-tests (musiques, artistes, titres).
  • J'ai écrit des requêtes SQL pour récupérer les données nécessaires à l'affichage des quiz et des blind-tests, ainsi que pour enregistrer les scores des participants.
  • J'ai optimisé les requêtes pour assurer une récupération rapide des données, même avec un grand nombre de quiz et de musiques.

Projet Capi (application de gestion de contenu) :

  • J'ai utilisé SQL pour créer une base de données qui stocke les informations des utilisateurs (comptes, permissions) et les données des capteurs et de la carte (mesures, positions, états).
  • J'ai écrit des requêtes SQL pour gérer les comptes utilisateurs, enregistrer les données des capteurs et de la carte, et récupérer ces données pour l'affichage et l'analyse.
  • J'ai mis en place des mécanismes de sécurité (comme le hachage des mots de passe) au niveau de la base de données pour protéger les informations sensibles.
